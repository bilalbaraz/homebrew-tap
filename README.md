# Bilal’s Homebrew Tap

Homebrew tap for shipping my CLI tools and apps.

## Install

```bash
brew tap bilalbaraz/tap
```

## Install Packages

```bash
# formula
brew install bilalbaraz/tap/<name>
```

```bash
brew update
brew upgrade

brew uninstall <formula>
```
